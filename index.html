Trading Day Toolkit

Upload historical prices (CSV) → rank tickers → get allocation + stops.

1) Competition Settings

Tip: use 1.6× exposure early; if you’re behind mid-month, consider 1.9×.

2) Upload Price CSVs

Upload one or more CSV files. Each CSV should contain daily prices for a single ticker. Accepted columns: Date and Close (or Adj Close). Header row required.

Where do I get CSVs?
  • Yahoo Finance → Ticker → Historical Data → Download
  • Any data source that provides Date + Close columns

3) Results

Upload CSVs and click Analyze.

4) Action Checklist (What to do in StockTrak)

  1. Every Friday after close: recompute ranks; hold the top N that pass trend filter (Close > MA20 > MA50).
  2. Use LIMIT orders for entries/exits (except emergency exits).
  3. Set stops: initial stop = X% below entry, then trailing stop ladder as you gain.
  4. Don’t overtrade: rebalance weekly unless stopped out.
